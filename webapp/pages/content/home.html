About

RasterModeler is a powerful and customizable entity relation schema modeler. Create editable schemas that can be easily embedded in html and markdown files. Export model to SQL statements using builtin scripts or create your own script according to your needs. This software has been provided free of charge, No login or registration required. You can help make this software better by submitting new scripts to project Github repository.

Start drawing

Start drawing now using the browser based web version click here.

The web version is subject to frequent changes. You can get a permanent stable Electron based version of the product for Windows and Linux on Downloads page.

Features

  • Create and edit tables
  • Preview support for creating and moving objects over the canvas
  • Camera minimap navigation
  • Zooming support
  • Undo/redo commands
  • Select tables and edit their name, type and special properties
  • Export to clipboard or save as .TXT or .PNG
  • Import .TXT files
  • View, manage and execute builtin scripts
  • Write your own scripts using builtin text editor
  • Share your scripts with others using Utterances comments
